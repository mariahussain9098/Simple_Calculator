# Simple_Calculator

A simple responsive calculator application built with HTML, CSS, and JavaScript.

## Features

- Addition, subtraction, multiplication, and division operations.
- Percentage calculation.
- Clear and backspace functionality.

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap 4.5.2
